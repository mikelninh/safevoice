NEXT STEP

AI classification — the plan

Goal: Victim pastes text → AI returns severity, offense categories, and applicable German laws as structured JSON

Approach: 3-tier classifier with fallback

Tier 1: OpenAI GPT-4o-mini — best accuracy, understands context and sarcasm
Tier 2: HuggingFace transformer — works offline, no API key
Tier 3: Regex patterns — guaranteed fallback, never fails

Prompt engineering plan:

1. System prompt with legal expert persona
2. Category enums (harassment, threat, misogyny, scam...)
3. Law list (§ 185, § 241 StGB, NetzDG...)
4. JSON schema enforcement for structured output
5. Compare techniques: zero-shot vs. system prompt vs. JSON schema vs. few-shot

Working prototype already classifies using regex (Tier 3) — let me demo it
