AI ENGINEERING

3-tier classifier with fallback

SafeVoice never returns "analysis unavailable"

TIER 1: Claude API

Understands context, sarcasm, indirect threats. Maps to specific § StGB paragraphs. Writes bilingual summaries (DE+EN).

Structured JSON output via prompt engineering

TIER 2: Transformer

HuggingFace toxic-comment-model. Works offline. Multilingual toxicity scoring. No API key needed.

TIER 3: Regex

Keyword patterns in DE, EN, Turkish, Arabic. Zero dependencies. Instant. Guaranteed fallback.

Prompt engineering: system prompt with legal expert persona + category enums + law list + JSON schema enforcement
Compared 4 techniques: zero-shot → system prompt → JSON schema → few-shot. JSON schema won.

TECH STACK
